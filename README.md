## Project Objective
To extract actionable insights from Zameen.com property listings such as:
- Pricing trends
- Neighborhood comparisons
- Listing quality
- Investor decision support

## Dataset
The dataset is sourced from Zameen.com via Google Sheets:  
[Dataset Link](https://docs.google.com/spreadsheets/d/12JWizEwTPYgXOmh_762iy-Q5694zxhlO/edit?gid=2004884327)

## Deliverables
1. Jupyter Notebook with end-to-end analysis: `zameen_eda.ipynb`  
2. Executive summary of key insights (see below).

## Key Insights
- Most expensive city: Karachi (example)  
- Least expensive city: Multan (example)  
- Top 10 expensive locations include DHA, Clifton, Gulberg, etc.  
- Bedrooms with highest average price: 5+  
- Correlation between price and area is strong (~0.8)

## How to Run
1. Clone the repository  
2. Open the notebook in Jupyter or Google Colab  
3. Install required libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`  
4. Run all cells to reproduce the analysis
